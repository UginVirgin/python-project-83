# Page Analyzer
Page Analyzer — это веб-приложение для анализа SEO-пригодности веб-страниц, построенное на Flask. Проект позволяет проверять базовые мета-теги страниц и отслеживать историю проверок.

## О проекте
Page Analyzer анализирует указанные страницы и извлекает базовую SEO-информацию. Приложение сохраняет историю проверок для каждого URL, позволяя отслеживать изменения в мета-тегах с течением времени. Это учебный проект, демонстрирующий работу с Flask, PostgreSQL, логированием и веб-парсингом.

### Основной функционал
- ✅ **Добавление URL** для анализа с валидацией корректности
- ✅ **Нормализация URL** — автоматическое приведение к единому формату (схема + домен)
- ✅ **Защита от дубликатов** — повторное добавление URL показывает информационное сообщение
- ✅ **Парсинг SEO-метаданных**:
  - Заголовок страницы (title)
  - Заголовок первого уровня (h1)
  - Мета-описание (description)
  - Код ответа HTTP (status code)
- ✅ **История проверок** для каждого URL с сортировкой по дате
- ✅ **Обрезка длинных полей** — поля h1, title, description обрезаются до 255 символов с добавлением "..."
- ✅ **Список всех URL** с отображением последнего статуса и даты последней проверки

## Установка

### 1. Клонирование репозитория

```bash
git clone https://github.com/UginVirgin/python-project-83.git
cd python-project-83
```

### 2. Настройка виртуального окружения и зависимостей

```bash
make install
```

### Запуск

```bash
poetry run flask --debug --app page_analyzer:app run
```
или
```bash
make dev
```

### Сборка деплоя
```bash
./build.sh
```
или
```bash
make build
```

## Команды Makefile
